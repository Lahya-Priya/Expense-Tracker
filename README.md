# 💸 Expense Tracker Web App

A simple, lightweight, and responsive **Expense Tracker** that allows users to log in, track their income and expenses, and view their financial summary — all in the browser with `localStorage` persistence.



## 🚀 Features

🔐 Login/Signup System (using only `username` & `password`).
📈 Track Income and Expenses.
🧾 Editable and Deletable Transactions.
💰 Real-time Balance Calculation.
🌐 Persistent Data using LocalStorage.
🎨 Clean, Responsive UI with CSS.


## 🧠 Technologies Used

- **HTML**
- **CSS3**
- **JavaScript**
- **LocalStorage API**



## 🔧 How to Run

1. **Download or clone** the repository:

    **bash**
   git clone https://github.com/Lahya-Priya/expense-tracker.git

2. Open `signup.html` in your browser and **create an account**.
3. After logging in, you'll be redirected to the main dashboard (`index.html`) to:
   - Add income or expenses
   - View your balance and transaction history
   - Edit or delete any transaction
4. **Logout** anytime using the top-right logout button.
